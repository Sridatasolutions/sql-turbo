# 🧩 Mission 2 Quiz — Sorting & Filtering Mastery

Answers at the bottom.

---

**1.** Default sort direction of `ORDER BY`?
- A) DESC
- B) ASC
- C) Random
- D) By insertion order

**2.** How to sort by salary high-to-low?
- A) `ORDER BY salary ASC`
- B) `ORDER BY salary DESC`
- C) `SORT salary DESC`
- D) `ORDER salary DOWN`

**3.** Which returns distinct department values only?
- A) `SELECT UNIQUE department_id ...`
- B) `SELECT DISTINCT department_id ...`
- C) `SELECT department_id GROUP`
- D) `SELECT ONLY department_id ...`

**4.** `WHERE status IN ('Active','On Leave')` is equivalent to:
- A) `status = 'Active' AND status = 'On Leave'`
- B) `status = 'Active' OR status = 'On Leave'`
- C) `status BETWEEN 'Active' AND 'On Leave'`
- D) `status LIKE 'Active'`

**5.** How to page results 11–20 of a sorted list?
- A) `LIMIT 20`
- B) `LIMIT 10 OFFSET 10`
- C) `OFFSET 20`
- D) `LIMIT 11 TO 20`

**6.** Which finds employees NOT in department 3?
- A) `WHERE department_id != 3`
- B) `WHERE department_id = NOT 3`
- C) `WHERE NOT department_id`
- D) `WHERE department_id <> '3' ONLY`

**7.** `ORDER BY department_id, salary DESC` sorts how?
- A) Salary first, then department
- B) Department ascending, then salary descending within each
- C) Both descending
- D) Both ascending

**8.** Which pattern finds names starting with "A"?
- A) `LIKE '%A'`
- B) `LIKE 'A%'`
- C) `LIKE '_A'`
- D) `= 'A'`
